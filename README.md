# Algoritmo de Dijkstra em C

Este repositório contém a implementação prática do Algoritmo de Dijkstra desenvolvida na linguagem C em arquivo único (main.c). 
O objetivo do projeto é encontrar o caminho mínimo (menor custo) de um vértice de origem para todos os outros nós em um grafo ponderado com pesos não-negativos.

O projeto utiliza uma Matriz de Adjacência para a representação do grafo e foi testado utilizando o ambiente de terminal **Git Bash**.

# Como Compilar

Para rodar este projeto na sua máquina, você precisa ter um compilador C instalado (como o GCC). 
Siga os passos abaixo no seu terminal:

### 1. Clonar o repositório ou entrar na pasta
Se você baixou os arquivos, abra o seu terminal (recomenda-se o Git Bash) e navegue até a pasta do projeto:
```bash
cd C:\Caminho\Ate\A\Pasta\Algoritmo-Grafos
```
###  2. Compilar o código
Use o GCC para ler o arquivo main.c e gerar o executável do programa:

```bash
gcc main.c -o programa
```

### 3. Executar o programa
No Git Bash (ou terminais estilo Linux), execute o comando utilizando a barra /:

```bash
./programa.exe
```
(Caso esteja usando o Prompt de Comando padrão do Windows, use apenas programa.exe ou .\programa.exe)
